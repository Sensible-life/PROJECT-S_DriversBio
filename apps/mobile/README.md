# Mobile App

## Purpose

Primary Flutter app for DriveAI.

## Current status

- Android and iOS Flutter project scaffolded in `apps/mobile`
- Initial app shell reflects the driving study workflow
- API integration is still the next step

## Run

```bash
cd apps/mobile
flutter run
```
